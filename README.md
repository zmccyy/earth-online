# 地球online - 文本版

> 现实生活模拟器，体验真实的人生模拟！

## 📖 项目简介

地球online是一款基于Python开发的文本模拟游戏，以幽默的方式模拟现实生活。玩家可以选择不同的身份，完成任务，管理资源，体验"地球online"这个网络梗的乐趣。

### 游戏特色

- 🎭 **多种身份选择**：学生、上班族、科学家、探险家，每种身份有独特加成
- 📋 **丰富任务系统**：15个日常任务，包含通用任务和身份专属任务
- 🎲 **随机事件系统**：15个随机事件，增加游戏趣味性和不确定性
- 📊 **属性管理系统**：智力、体力、社交、财富等多维度属性成长
- 💾 **完整存档系统**：支持游戏存档和读取
- 🎨 **精美文本界面**：带边框和格式化的文本UI

## 🎮 游戏玩法

### 开始游戏

1. 运行 `python main.py` 启动游戏
2. 选择"开始新游戏"
3. 输入角色名称
4. 选择身份（每种身份有不同加成）：
   - **学生**：智力+5，擅长学习
   - **上班族**：财富+500，擅长赚钱
   - **科学家**：智力+10，擅长研究
   - **探险家**：体力+10，擅长探索

### 游戏循环

在游戏中，你可以：
- **执行任务**：完成任务获得奖励（属性、资源、经验）
- **查看背包**：查看已获得的物品
- **查看技能**：查看已学会的技能
- **查看成就**：查看已完成的成就
- **保存游戏**：保存当前进度
- **退出游戏**：结束游戏

### 属性系统

**基础属性**：
- 🧠 **智力**：影响学习和研究效率
- 💪 **体力**：影响运动和探险能力
- 🗣️ **社交**：影响社交活动效果
- 💰 **财富**：游戏中的金钱

**资源属性**：
- ⚡ **精力**：执行任务需要消耗精力
- 🍖 **饱食度**：需要定期进食维持
- 😊 **快乐值**：影响游戏体验

### 任务系统

**通用任务**：
- 🍳 吃早餐/午餐/晚餐 - 恢复饱食度
- 💪 锻炼身体 - 提升体力
- 🎉 社交活动 - 提升社交和快乐值
- 😴 休息放松 - 恢复精力
- 💼 兼职打工 - 赚取金钱
- 📚 阅读书籍 - 提升智力
- 🎬 看电影 - 提升快乐值
- 🛍️ 逛街购物 - 提升快乐值和社交
- 🛏️ 睡觉 - 大幅恢复精力

**身份专属任务**：
- 👔 **上班族**：上班打卡（赚钱、经验）
- 🎓 **学生**：学习充电（提升智力、经验）
- 🔬 **科学家**：科研项目（提升智力、赚钱、经验）
- 🧭 **探险家**：野外探险（提升体力、快乐值、经验）

游戏中会随机触发各种事件：

**正面事件**：
- 💰 意外之财 - 捡到钱包

- 🌟 幸运日 - 一切顺利

## 🛠️ 技术架构

### 技术栈

- **编程语言**：Python 3.9+
- **数据格式**：JSON
- **界面**：命令行文本界面

### 项目结构

```
地球online/
├── main.py              # 游戏入口
├── player.py            # 玩家系统
├── ui.py                # 界面系统
├── tasks.py             # 任务系统
├── events.py            # 事件系统
├── data/                # 数据文件夹
│   ├── tasks.json       # 任务数据
│   └── events.json      # 事件数据
├── saves/               # 存档文件夹

```

### 核心模块

#### 1. Player（玩家系统）
- 角色创建和属性管理
- 身份加成系统
- 资源管理

#### 2. TaskSystem（任务系统）
- 任务数据加载
- 任务执行逻辑
- 资源检查和消耗

#### 3. EventSystem（事件系统）

- 随机事件触发
- 事件效果应用

#### 4. UI（界面系统）
- 文本界面显示
- 菜单系统
- 状态显示
- 消息提示

## 📊 游戏数据

### 任务数据格式

```json
{
  "id": "task_id",
  "name": "任务名称",
  "description": "任务描述",
  "requirements": {
    "identity": "worker"
  },
  "costs": {
    "energy": 30,
    "hunger": 20
  },
  "rewards": {
    "wealth": 200,
    "experience": 10
  },
  "time_cost": 480
}
```

### 事件数据格式

```json
{
  "id": "event_id",
  "name": "事件名称",
  "description": "事件描述",
  "probability": 0.05,
  "effects": {
    "wealth": 100,
    "happiness": 10
  }
}
```

## 🎯 游戏策略

### 新手建议

1. **选择合适的身份**：
   - 想快速赚钱：选择上班族
   - 想提升属性：选择学生或科学家
   - 想体验探险：选择探险家

2. **平衡资源管理**：
   - 定期进食维持饱食度
   - 合理安排休息恢复精力
   - 适度社交提升快乐值

3. **任务优先级**：
   - 优先完成身份专属任务（收益更高）
   - 平衡赚钱和属性提升
   - 注意资源消耗，避免资源枯竭

4. **应对随机事件**：
   - 保持一定的财富储备应对负面事件
   - 正面事件是额外收益，不要过度依赖

## 🔧 扩展开发

### 添加新任务

在 `data/tasks.json` 中添加新任务：

```json
{
  "id": "new_task",
  "name": "新任务",
  "description": "任务描述",
  "costs": {"energy": 20},
  "rewards": {"wealth": 100},
  "time_cost": 120
}
```

### 添加新事件

在 `data/events.json` 中添加新事件：

```json
{
  "id": "new_event",
  "name": "新事件",
  "description": "事件描述",
  "probability": 0.05,
  "effects": {"happiness": 10}
}
```

### 添加新身份

1. 在 `player.py` 的 `apply_identity_bonus()` 方法中添加新身份
2. 在 `ui.py` 的 `get_identity_name()` 方法中添加中文名称
3. 在 `data/tasks.json` 中添加身份专属任务
